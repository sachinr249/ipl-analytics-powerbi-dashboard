## IPL Analytics Dashboard (2008–2024)
Project Overview

- This project presents an interactive Power BI dashboard built using ball-by-ball IPL data (2008–2024) to analyze team performance, player statistics, venue trends, match insights, and historical IPL records.

- The dashboard transforms over 260,000+ deliveries and 1,095 IPL matches into an intuitive analytics experience for cricket enthusiasts and decision-makers.
