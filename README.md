# IPL Analytics Dashboard (2008–2024)
- This project presents an interactive Power BI dashboard built using ball-by-ball IPL data (2008–2024) to analyze team performance, player statistics, venue trends, match insights, and historical IPL records.

- The dashboard transforms over 260,000+ deliveries and 1,095 IPL matches into an intuitive analytics experience for cricket enthusiasts and decision-makers.

## Dashboard Features
### Executive Overview
- KPI Cards
- Season Trends
- Team Wins
- Run Distribution
### Team Analytics
- Team Comparison
- Win Percentage
- Venue Performance
- Head-to-Head Analysis
### Batter Analytics
- Career Runs
- Strike Rate
- Boundary Analysis
- Phase-wise Performance
### Bowler Analytics
- Wickets
- Economy
- Dot Balls
- Dismissal Analysis
### Venue Analytics
- Average Scores
- Toss Impact
- Venue Performance
### Match Insights
- Match Summary
- Scorecard
- Run Progression
### Records & Milestones
- Most Runs
- Most Wickets
- Fastest Fifties
- Highest Team Totals

## Power BI Features Used
- Data Modeling (Star Schema)
- Power Query
- DAX Measures
- Drillthrough Page
- Navigation Buttons
- Bookmarks
- Dynamic Titles
- Custom Tooltips
- Field Parameters
- Conditional Formatting
- Interactive Slicers
